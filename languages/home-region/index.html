Home Region

Forgotten Realms Campaign Setting distributes automatic languages in reference to a character's home region. See Table 1-4: Character Regions on page 30 and page 9 of the Forgotten Realms Campaign Setting for more details.

Living Languages which are available as home region language include (FRCS, p. 85):
